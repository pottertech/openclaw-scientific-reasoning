# Changelog

All notable changes to OpenClaw-Scientific-Reasoning will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] - 2026-04-14

### Added
- Initial release
- Core scientific reasoning toolkit (SymPy, SciPy, OR-Tools, Z3, CVXPY, PyMC, RDKit, Cantera, PySCF, Lcapy, SageMath)
- Installation and uninstallation scripts
- Update lifecycle with safe and refresh-lock modes
- Verification tooling with import checks and smoke tests
- Capability metadata for all reasoning domains
- Agent playbooks for all major tools
- Router hints for domain routing
- Wrapper contracts and templates
- Integration guide and startup brief snippet
- Test scaffolding (smoke, routing, wrappers, integration)
- Makefile for common operations
